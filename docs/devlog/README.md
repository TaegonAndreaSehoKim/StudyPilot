# StudyPilot Devlog

This folder stores weekly development logs for StudyPilot.

The format follows the existing project documentation style used in earlier work:

- each week has its own file
- each day or milestone records focus, what changed, validation, and result
- weekly snapshots summarize current state, quality checkpoints, and next steps

Current logs:

- `week1.md`: planning, GitHub setup, MVP backend, mobile app foundation, tests, and documentation
