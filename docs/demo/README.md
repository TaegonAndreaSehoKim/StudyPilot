# Demo Materials

This folder contains small source-grounded files for repeatable StudyPilot demos.

Suggested demo flow:

1. Create a course named `OMSCS AI`.
2. Create a section such as `Midterm 1`.
3. Add `omscs_ai_sample_notes.md` to that section.
4. Generate Additional Explanation or Quick Review notes from the source.
5. Generate flashcards.
6. Generate a quiz.
7. Intentionally miss one or two quiz questions.
8. Confirm weak topics appear on the dashboard.

The sample notes are intentionally short so fake-AI generation remains quick and predictable. Use section-level tools during demos when you want to show multiple sources grouped by unit, chapter, midterm, or final scope.

For a full Expo Go checklist, see [mobile_walkthrough.md](mobile_walkthrough.md).

For a short presentation-oriented script, see [demo_script.md](demo_script.md).
