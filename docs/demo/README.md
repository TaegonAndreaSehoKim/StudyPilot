# Demo Materials

This folder contains small source-grounded files for repeatable StudyPilot demos.

Suggested demo flow:

1. Create a course named `OMSCS AI`.
2. Upload `omscs_ai_sample_notes.md` from the course detail screen.
3. Generate a concise summary.
4. Generate flashcards.
5. Generate a quiz.
6. Intentionally miss one or two quiz questions.
7. Confirm weak topics appear on the dashboard.

The sample notes are intentionally short so fake-AI generation remains quick and predictable.

For a full Expo Go checklist, see [mobile_walkthrough.md](mobile_walkthrough.md).
